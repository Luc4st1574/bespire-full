Posibles Soluciones

Opción 1: Actualizar Node.js (Recomendado)

# Usando nvm (si está instalado)
nvm install 18
nvm use 18

# O descargar desde nodejs.org
# Instalar Node.js 18 LTS o superior

Opción 2: Downgrade temporal de dependencias

# Cambiar en package.json:
"next": "14.2.15",
"tailwindcss": "^3.4.0"

# Y ejecutar:
npm install

Opción 3: Usar colores inline mientras tanto

Usar style={{backgroundColor: 'var(--color-brand-dark)'}} en lugar de className="bg-brand-dark"
